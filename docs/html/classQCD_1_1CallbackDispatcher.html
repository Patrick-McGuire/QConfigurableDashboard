Detailed Description

CallbackDispatcher provides a module to register and trigger callbacks based on string identifiers.

Callbacks are standard functions, and must have 1 argument of type QCD::Json. Alternatively ID Callbacks can be used, which are called regardless of the identifier triggered. ID Callbacks are instead passed 2 arguments (must be of types QCD::Json, std::string). The string will be the identifier. This is primarily used by QCD::ThreadedModule to tunnel events into it's thread. This class is intended to be used as a parent class, and is derived by QCD::AppManager and QCD::ThreadedModule.
