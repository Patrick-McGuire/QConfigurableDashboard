Detailed Description

Base class for all QCD custom widget.

This class handles all root functionality for QCD's custom widgets. This includes dragging, themes and periodically being updated. This class should be the parent class for all custom widgets used in a QCD application.

Note
m_appManager is not guaranteed to be populated before QCD::Widget::run() is called

◆ setUpdateRateScale()

bool QCD::Widget::setUpdateRateScale ( int  a_scale)
virtual

Sets a multiplier on the application update rate specific to this widget.

This will update this widget every nth QCD::Widget::smartUpdate() call. This can be used to improve performance on widgets that don't need to be updated frequently. This may be re-implemented by derived classes, so it's behavior is not guaranteed on all derivations of QCD::Widget.
