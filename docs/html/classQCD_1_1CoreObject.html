Provides a callback and data interface to all QCD classes.

Triggers all callbacks registered to the specified identifier.

All callbacks registered with QCD::CallbackDispatcher::registerCallback(), with the same identifier, will be called and passed the specified Json data. This also triggers all tunnel Callbacks.

Parameters
a_identifierString identifier to trigger
a_jsonJson data to pass
a_excludeIdIndexIndex in vector of callbacks to skip

Implements QCD::CallbackDispatcherInterface.

Reimplemented in QCD::ThreadedModule.
