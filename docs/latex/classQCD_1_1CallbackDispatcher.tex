\doxysubsection{Detailed Description}
\mbox{\hyperlink{classQCD_1_1CallbackDispatcher}{Callback\+Dispatcher}} provides a module to register and trigger callbacks based on string identifiers. 

Callbacks are standard functions, and must have 1 argument of type \mbox{\hyperlink{namespaceQCD_ad0c1bf8c2985617119593554d8b1db33}{QCD\+::\+Json}}. Alternatively tunnel Callbacks can be used, which are called regardless of the identifier triggered. tunnel Callbacks are instead passed 2 arguments (must be of types \mbox{\hyperlink{namespaceQCD_ad0c1bf8c2985617119593554d8b1db33}{QCD\+::\+Json}}, std\+::string). The string will be the identifier. This is primarily used by \mbox{\hyperlink{classQCD_1_1ThreadedModule}{QCD\+::\+Threaded\+Module}} to tunnel events into it\textquotesingle{}s thread. This class is intended to be used as a parent class, and is derived by QCD\+::\+App\+Manager and \mbox{\hyperlink{classQCD_1_1ThreadedModule}{QCD\+::\+Threaded\+Module}}. \begin{DoxySeeAlso}{See also}
