\doxysubsection{Detailed Description}
Base class for all \mbox{\hyperlink{namespaceQCD}{QCD}} containers. 

The goal of \mbox{\hyperlink{namespaceQCD}{QCD}} containers is to remove the concept from QT of widgets having layouts to encompass other widgets. Instead with \mbox{\hyperlink{namespaceQCD}{QCD}}, layouts are just another type of \mbox{\hyperlink{classQCD_1_1Widget}{QCD\+::\+Widget}} that you can add children directly too. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classQCD_1_1Widget}{QCD\+::\+Widget}} 
\end{DoxySeeAlso}
