\doxysubsection{Detailed Description}
Base class for all \mbox{\hyperlink{namespaceQCD}{QCD}} custom widget. 

This class handles all root functionality for \mbox{\hyperlink{namespaceQCD}{QCD}}\textquotesingle{}s custom widgets. This includes dragging, themes and periodically being updated. This class should be the parent class for all custom widgets used in a \mbox{\hyperlink{namespaceQCD}{QCD}} application. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classQCD_1_1Container}{QCD\+::\+Container}} These are \mbox{\hyperlink{namespaceQCD}{QCD}}\textquotesingle{}s re-\/implementations of QLayouts 
\end{DoxySeeAlso}
